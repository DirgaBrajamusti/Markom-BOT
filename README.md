# Whatsapp Bot
## Installation

 - Install node package 
 - Install python dependencies 
 - Install Redis
 - Install MySQL/MariaDB

## Usage
 - Run the whatsapp-web.js using `node app.js`
 - Scan the QR Code
 - Run python using `python bot.py`
